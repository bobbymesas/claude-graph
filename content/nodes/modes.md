Claude Code can be run in several modes: Interactive for conversational coding, Plan Mode for proposal-first workflow, Headless for CI/CD automation, and Agent Mode for autonomous multi-agent orchestration.

### EXAMPLE: When to use each mode
### FILE: Mode overview
### LANG: plaintext

```plaintext
Interactive  claude            Default — conversational coding
Plan Mode    /plan             Propose before building
Headless     claude -p "..."   CI/CD, scripts, automation
Agent Mode   Task tool         Parallel subagent orchestration
```
